# Contributing to the Dataset

Thank you for contributing to the Ambient Clinical Scribe dataset.

## Before adding a new condition

Please ensure that every clinical condition includes:

- One transcript
- One expected SOAP note
- ICD-10 mapping (if applicable)
- Audio sample (optional)

---

## Commit Messages

Examples:

Added asthma transcript and SOAP test case

Added migraine transcript and SOAP test case

Expanded ICD-10 dataset

---

## Naming Rules

Transcript

condition_case_01.txt

SOAP

condition_case_01_expected.txt

Audio

condition_case_01.wav

---

## Quality Standards

- Use realistic doctor-patient conversations.
- Keep medical terminology accurate.
- Follow the SOAP format consistently.
- Ensure filenames follow the naming convention.
